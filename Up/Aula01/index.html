Então, o que é HTML?

HTML (Linguagem de Marcação de Hipertexto)1 é o código que você usa
para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo
pode ser estruturado em parágrafos, em uma lista com marcadores
ou usando imagens e tabelas.
